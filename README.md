# openurl-cli

A simple cli for open a url from a command line.

This is helpful to open urls in npm scripts.

## Usage

Install:

```
npm install --global openurl-cli
```

or

```
npm install --dev openurl-cli
```

Run the following command to open a url:

```
openurl-cli <URL>
```
